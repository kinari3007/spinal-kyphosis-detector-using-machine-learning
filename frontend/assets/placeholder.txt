Assets Directory for Kyphosis Prediction App

This directory is intended to contain medical illustrations and diagrams:

1. spine-comparison.png - Comparison between normal and kyphotic spine curvature
2. kyphosis-xray.png - X-ray image showing kyphotic spine with curve measurements
3. posture-illustration.png - Illustration showing postural progression to kyphosis

The frontend HTML includes fallback displays when these images are not available,
showing placeholder cards with descriptive text instead.

For a production version, these would be replaced with actual medical illustrations
or diagrams created by medical illustrators or sourced from appropriate medical
image libraries with proper licensing.
